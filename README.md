# Late Delivery Prediction Project

## 📌 Overview
This project predicts whether a delivery will be late using Machine Learning.

## 📊 Dataset
The project uses the APL Logistics dataset which contains:
- Order details
- Shipping mode
- Product price
- Customer information
- Delivery time

Due to the large size of the dataset (~60MB), it is not uploaded in this repository.

## 🤖 Model Used
- Random Forest Classifier

## ⚙️ Process
- Data preprocessing
- Handling missing values
- Converting categorical data to numerical
- Model training and testing

## 📈 Output
The model predicts late delivery with high accuracy.

## 🚀 Conclusion
Machine learning helps in identifying delivery risks and improving supply chain efficiency.
